Summary: Latitude vs. X

The aim of the project was to observe and analyze changes in the weather as you approach the equator. Using data from OpenWeatherMap API to assemble a dataset on over 500 cities, the following visualizations demonstrate weather patterns.

We can see that the weather does indeed follow certain patterns. Mainly, as we suspected that the close one approaches the equator, the warmer the weather. It does appear that the southern hemisphere is significantly warmer for the same time of year as the northern hemisphere.
